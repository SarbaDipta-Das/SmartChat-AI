SmartChat AI is a full-stack GenAI web application built with FastAPI and Llama 3 (via Groq API) that enables users to upload documents and receive AI-powered answers grounded in their own content using a custom RAG (Retrieval-Augmented Generation) pipeline. The application supports 8 specialized domains including Banking, Finance, Healthcare, E-Commerce, Skincare, Cooking, Education, and General knowledge — each powered by carefully engineered system prompts. Users can interact via text or voice, with browser-native speech recognition built in. The project also features a complete AI Interview Platform where users upload their resume, receive personalized behavioral and coding interview questions, get real-time AI feedback and scores, write code in a Monaco Editor environment, and receive a final performance report — all within the same application. A computer vision-based proctoring system monitors the session for face visibility and phone usage using the Canvas API. The entire frontend is built in vanilla HTML, CSS, and JavaScript with no frameworks, while the backend handles async document ingestion, BM25-based chunk retrieval, and Groq API calls through FastAPI.
